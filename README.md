# Pin Configuration

Basically im using standard SPI Pin

| ESP32 | RC522  |
|-------|--------|
| 5     | SS/SDA |
| 15    | IRQ    |
| 21    | RST    |
| 18    | SCK    |
| 23    | MOSI   |
| 19    | MISO   |
